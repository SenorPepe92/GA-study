static (default):
the position is whatever the browser calculates for the this element in the page flow. You cant change it with top/left CSS offsets
relative:
Offstes do work and are calculated realtive to the the default (static) position in browsers page flow.
Parent of absolute
absolute:
Offsets work and are calutated relative to the nearest containing element (ancestor) which has a specifc non-static position value.. otherwise it is relative the body tag i.e. whole document
fixed:
Offsets are calutated relative to the entire browser window (viewport) - i.e. the element does NOT move when you scroll the document
